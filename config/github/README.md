### Github personal access tokens

The [GDSC tools](https://github.com/Geospatial-Digital-Special-Collections/tools) repository is a private repository for the time being. If you need to build the [GDSC repository docker container](https://github.com/Geospatial-Digital-Special-Collections/docker/tree/main/builds/repository) you will need a personal access token with read-only access to the [GDSC tools](https://github.com/Geospatial-Digital-Special-Collections/tools) repository.

Please reach out to gdsc@miami.edu to gain access to the [GDSC tools](https://github.com/Geospatial-Digital-Special-Collections/tools) and then create a personal access token with read only access following the [github instructions for managin personal access tokens](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens#creating-a-fine-grained-personal-access-token).

Once you have the token save it in the file "tools.pat" following the example in "tools.pat.example."
